# minuteur-examen-desktop
Minuteur pour des examens avec version html
This version you can download and run locally on any computer in a browser. If there is voice support on the computer (such as Mac or Windows) it can speak commands at different points (customisable).
You can change the logo in the countdown wheels.
You can customise the additional time options (so overtype the 25% with either a different percentage, a number (for a proportion, so 0.5 for example for an extra 50%), or 1/3 for an additional one third.
You can adjust the reading time. It can be turned off for individual exams.
Fine for one exam, but with two in the same room, they can have different times.
There are (current at the time of writing) spoken commands to match the IB requirements (not the full, long text, but the time warnings and start and end of exam warning — you'll have to do the full preamble manually and the end section, unless you really want to type it all in - make sure you select the type of IB exam and it should speak the correct commands for that type of exam).
There are two warnings (default 30 and 5 minutes that you can adjust).
You can go silent, just beeps at the key points, or spoken text.
The current time shows at the top.
Warning times are shown next to the main time remaining and cross off once completed.
If there are multiple exams, it says which exam the notification is for.
Add your own logo to replace the default one.
